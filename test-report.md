## 테스트 상세

### 테스트 1: 초기 빈 리스트 상태 표시
- **목적**: 아이템이 없을 때 빈 상태 안내 메시지가 표시되는지 확인
- **절차**: localStorage 초기화 후 페이지 로드
- **기대 결과**: "리스트가 비어 있습니다" 메시지 표시, 통계 미표시
- **실제 결과**: 기대와 동일

### 테스트 2: 버튼 클릭으로 아이템 추가
- **목적**: 입력 필드에 텍스트 입력 후 "추가" 버튼 클릭으로 아이템 추가
- **절차**: "우유" 입력 후 추가 버튼 클릭
- **기대 결과**: 리스트에 "우유" 표시, 빈 상태 메시지 제거, 통계 "0 / 1 완료", 입력 필드 초기화 및 포커스 유지
- **실제 결과**: 기대와 동일

### 테스트 3: Enter 키로 아이템 추가
- **목적**: 입력 필드에서 Enter 키로 아이템 추가 가능한지 확인
- **절차**: "계란" 입력 후 Enter 키 입력
- **기대 결과**: 리스트에 "계란" 추가, 통계 "0 / 2 완료"
- **실제 결과**: 기대와 동일

### 테스트 4: 빈/공백 입력값 추가 방지
- **목적**: 공백만 입력했을 때 아이템이 추가되지 않는지 확인
- **절차**: 공백 문자열("   ") 입력 후 Enter
- **기대 결과**: 아이템 추가되지 않음, 리스트 변화 없음
- **실제 결과**: 기대와 동일

### 테스트 5: 체크박스 토글 (완료 표시)
- **목적**: 체크박스 클릭 시 아이템이 완료 상태로 전환되는지 확인
- **절차**: "우유" 체크박스 클릭
- **기대 결과**: 체크박스 checked 상태, 통계 "1 / 2 완료"
- **실제 결과**: 기대와 동일

### 테스트 6: 체크 해제 (토글 복원)
- **목적**: 체크된 아이템을 다시 클릭하면 해제되는지 확인
- **절차**: 체크 상태인 "우유" 체크박스 재클릭
- **기대 결과**: 체크 해제, 통계 "0 / 2 완료"
- **실제 결과**: 기대와 동일

### 테스트 7: 아이템 삭제
- **목적**: 삭제 버튼(✕) 클릭 시 아이템이 제거되는지 확인
- **절차**: "계란" 삭제 버튼 클릭
- **기대 결과**: "계란" 제거, 리스트에 "우유"만 남음, 통계 "0 / 1 완료"
- **실제 결과**: 기대와 동일

### 테스트 8: 모든 아이템 삭제 후 빈 상태 복원
- **목적**: 모든 아이템 삭제 시 빈 상태 메시지가 다시 나타나는지 확인
- **절차**: 마지막 아이템 "우유" 삭제
- **기대 결과**: "리스트가 비어 있습니다" 메시지 표시, 통계 미표시
- **실제 결과**: 기대와 동일

### 테스트 9: LocalStorage 영속성 (새로고침)
- **목적**: 페이지 새로고침 후에도 데이터가 유지되는지 확인
- **절차**: "빵" 추가 후 페이지 새로고침
- **기대 결과**: 새로고침 후에도 "빵" 아이템 유지, 통계 정상 표시
- **실제 결과**: 기대와 동일

### 테스트 10: 다수 아이템 + 복합 통계 정확성
- **목적**: 여러 아이템의 체크 상태가 통계에 정확히 반영되는지 확인
- **절차**: 3개 아이템(빵, 사과, 생수) 중 2개(빵, 생수) 체크
- **기대 결과**: 통계 "2 / 3 완료"
- **실제 결과**: 기대와 동일

### 테스트 11: 중간 아이템 삭제 후 정합성 유지
- **목적**: 중간 위치 아이템 삭제 시 나머지 아이템의 체크 상태가 유지되는지 확인
- **절차**: 체크되지 않은 중간 아이템 "사과" 삭제
- **기대 결과**: 빵(checked), 생수(checked) 유지, 통계 "2 / 2 완료"
- **실제 결과**: 기대와 동일
